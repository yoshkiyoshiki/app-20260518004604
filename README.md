# 🌊 Flowspace

チームの集中を守りながら会話をナレッジに変換する、AIネイティブ・コミュニケーションプラットフォーム。

## 特徴

- **プロジェクトハブ** — チャンネルではなくプロジェクト単位で会話を整理。フェーズ管理（Active / Paused / Archived）対応
- **AIナレッジエンジン** — 会話から決定事項・TODO・Q&A・インサイトを自動抽出しナレッジベースに蓄積
- **フォーカスガード** — メッセージの緊急度を4段階で自動判定。集中モードで不要な通知をブロック
- **セマンティック検索 & タイムトラベル** — 自然言語検索と任意時点のプロジェクト状態再現
- **会話サマリー** — 任意期間の会話要約をワンクリック生成

## セットアップ

1. リポジトリをクローン
2. 依存関係をインストール: `pip install -r requirements.txt`
3. アプリを起動: `streamlit run main.py`

## 使い方

1. サイドバーからプロジェクトを作成（またはデモプロジェクトを生成）
2. チャットタブでメッセージを送信
3. ナレッジベースタブで自動抽出された決定事項・TODOを確認・承認
4. 検索タブで過去の会話を横断検索
5. フォーカスモードをONにして集中時間を確保
